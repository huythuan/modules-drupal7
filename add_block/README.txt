
-- SUMMARY --

The module allow you to add dynamic javascript code or dynamic block code or both to the page at a selected region.
This module is good for add block to a page with a selected region without using block module.
It is good for add a block for advertising for example,
This module is easy to use, easy to manage block

Main features:
- Add dynamic block to a page in a selected region
- Add dynamic Javascript to a page 
- Can Enable or Disable a block as well as javascript code
- Easy to manage block code


-- REQUIREMENTS --

PHP 5.


-- INSTALLATION --

* Put the module in your drupal modules directory and enable it in 
  admin/modules. 


-- CONFIGURATION --

* Configure user permissions in Administration » People » Permissions:
Go to admin/people/permissions and grant permission to any roles
that need to be able to add and edit block.



* Add a block code:
Just follow the admin/config/content/add-block/add and fill out the form.
